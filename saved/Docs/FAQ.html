FAQ - Read This Before You Ask

Does Stream Connector run in the cloud?

No.

Nothing runs in the cloud. Nothing phones home. Nothing listens on the internet.

Everything runs locally on your machine. If it’s talking to something, you told it to.

Do I need all the features?

Nope.

Every system is modular:

  • Use only VRChat
  • Use only TikTok
  • Use only Streamer.bot
  • Use only OwO
  • Use only PiShock
  • Use only Intiface
  • Use Everything

Nothing forces you into anything.

Can I use only VRChat?

Yes.

OSC works standalone. No TikTok. No bots. No nonsense.

Can I use only TikTok?

Yes.

TikTok → Stream Connector → Actions. No VRChat required.

Can VRChat touch drive my toys?

Yes.

SPS / OGB contacts feed the live-touch engine. Get touched in game and it drives your devices in real time - Intiface, OSC, PiShock, and the OwO vest, all at once, scaled to how much you're touched. Let go and it stops.

Do I need the Extended / Pro / Whatever key?

No.

That system is dead.

Everything runs unlocked now. No tiers. No gating. No bullshit.

Is this a subscription?

No.

There is no forced subscription. No background billing. No surprise charges.

Future add-ons (like advanced hardware integrations) may exist - but the core system stays free and local.

Is this safe?

Yes - and more importantly, it’s predictable.

  • Local-only execution
  • No inbound ports
  • No remote control
  • No telemetry
  • No background services
  • No silent updates

If something happens, it’s because you told it to.

Can this brick my system?

No.

Worst case? You misconfigure something and it doesn’t trigger. That’s it.

There is no persistent system modification. No kernel hooks. No driver installs.

Can I break stuff?

Yes - if you’re trying to.

But the system is designed with:

  • Rate limits
  • Cooldowns
  • Safety gates
  • Hard execution caps

You have control - not chaos.

Where does my data go?

Nowhere.

Everything lives inside: 

Stream Connector/
└── saved/
    ├── config/
    ├── controls/
    ├── logs/
    └── userdata/

Delete that folder and it’s a clean slate.

Final Answer

If you want:

  • Cloud dashboards
  • Subscriptions
  • Telemetry
  • Hand-holding

This probably isn’t for you.

If you want:

  • Control
  • Precision
  • Local execution
  • No corporate leash

Then yeah. You’re in the right place.
