Stream Connector

🔗 The Ultimate Interactive Control Engine for TikFinity, Twitch, YouTube, Kick, TikTok, VRChat, and Shock Feedback

Stream Connector is a purpose-built, GUI-driven control system designed for advanced stream automation, avatar interaction, and real-time device control.

It integrates Python automation, VRChat OSC, TikTok Live events, PiShock, and a modular chain engine to provide precise, deterministic control over interactive streaming environments.

🌟 Features Summary

🎁 TikTok Live

  • Responds to likes, gifts, follows, subs, and comments
  • Event chaining, scaling logic, and manual triggering

🧩 Chain Logic Engine

  • Visual chain builder with delays, timers, and mixed execution modes
  • Supports OSC, PiShock, internal events, and fallback logic

🧠 VRChat OSC

  • Automatic parameter discovery
  • Override protection and safe-state logic

⚡ PiShock Support

  • Realtime WebSocket v2
  • Pattern + intensity control
  • Owned & shared device routing

😈 Intimate Control Layer

  • Intiface / Buttplug: vibrate, oscillate, constrict, inflate, linear depth (strokers), and rotation
  • Multi-actuator and per-device modes; Buttplug v4 with automatic v3 fallback
  • SPS / OGB live-touch: VRChat contact drives your devices in real time
  • OwO haptic vest with continuous live-touch playback
  • Deterministic execution

🌐 Multi-Platform Integration

  • Streamer.bot support
  • External API
  • Live reload + diagnostics
