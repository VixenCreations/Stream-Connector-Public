🧩 Installation

This is not a wizard. This is not a “next-next-finish” toy.

This is a real system that expects you to know how to unzip a file and click a button. If that sounds scary - stop now.

🧾 Requirements

  • Windows 10 or newer
  • Administrator access (for first launch)
  • VRChat (optional, only if using OSC)
  • TikTok account (optional)
  • PiShock / OwO devices (optional)

⬇ Download

Stream Connector is distributed directly. No storefronts. No DRM. No middlemen.

Download:
https://1drv.ms/f/c/d8576d046783ce52/…

Download the latest folder, extract it anywhere you want, and run the executable.

🚀 First Launch

  1. Extract the archive
  2. Run Stream Connector.exe
  3. Allow firewall access when prompted
  4. Let the app initialize its folders

That’s it. No activation. No login. No license key.

⚙️ Initial Configuration

  1. Open the dropdown menu
  2. Select Configuration
  3. Set only what you actually use

Optional:

  • TikTok username (case sensitive)
  • PiShock account + API key
  • OSC (VRChat)

If you don’t use something - leave it blank. Nothing breaks.

🤖 Streamer.bot Integration (Optional)

If you want automation, this is where the magic happens.

  1. Download Streamer.bot https://streamer.bot/downloads
  2. Extract and launch it
  3. Go to Servers / Clients → WebSocket Server
  4. Start server on 127.0.0.1:8080
  5. Stream Connector will auto-connect

You should see:

  • Connected ✔
  • Subscribed ✔
  • No errors ✔

📦 Importing Example Actions

  1. Go to Streamer Kit/Streamerbot Example
  2. Open the export file
  3. Copy everything
  4. Paste into Streamer.bot → Import
  5. Confirm the warning

Yes, it warns you. That’s normal. That’s security working as intended.

✅ Final Notes

  • Everything runs locally
  • No cloud services required
  • No accounts needed
  • No tracking

If it breaks, check the logs. If something doesn’t fire, you misconfigured it.

That’s not an insult - that’s how power tools work.

🔥 Install once. Configure once. Own the system.
