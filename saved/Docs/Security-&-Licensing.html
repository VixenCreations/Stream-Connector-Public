🔓 Security & Licensing

Let’s clear the air immediately.

There is no licensing system anymore. No keys. No tiers. No activations. No servers deciding what you’re “allowed” to do.

This project is free, local, and yours to control.

📜 License Model

Stream Connector operates under a permissive, MIT-style model.

  • ✔ Free to use
  • ✔ Free to modify
  • ✔ Free to redistribute
  • ✔ Free for personal or commercial projects

No DRM. No paywalls. No feature gates.

If you have the files, you have the software.

🔐 Security Philosophy

Security here is boring by design - because boring security is reliable security.

  • Local execution only
  • No background services
  • No cloud dependency
  • No telemetry
  • No forced updates

Nothing phones home. Nothing runs without you launching it. Nothing hides what it’s doing.

🌐 Network & Runtime Behavior

  • Explicit WebSocket connections only
  • No external listeners unless enabled
  • All ports and endpoints are visible
  • Fails closed, not open

If it’s talking to something, you configured it.

For full technical breakdown, see:
Security & Network Architecture

🛡️ Antivirus & Trust

All builds are scanned and validated through:

  • ✔ Windows Defender cloud reputation checks
  • ✔ Standard executable trust heuristics
  • ✔ No packing, no obfuscation

There is no telemetry submission. There is no remote scanning. There is no hidden updater.

If your antivirus complains, it’s because this software:

  • Controls hardware
  • Opens sockets
  • Does real work

Not because it’s malicious.

🚫 What This Does NOT Do

  • No data collection
  • No analytics
  • No accounts
  • No cloud login
  • No remote kill switches

You own your install. You own your data. You own the consequences.

🧠 Final Word

This project exists because people wanted tools - not permission slips.

You get the full system. You get full control. You get full responsibility.

If that makes you uncomfortable, that’s fine.

If it makes you excited -

Welcome. You’re exactly who this was built for.
