🤖 Streamer.bot Integration

Stream Connector includes a native, first-class integration with Streamer.bot. No plugins. No middleware. No duct tape.

It listens locally, parses events safely, and executes only what you explicitly allow.

🔌 Connection Overview

ProtocolWebSocket
Addressws://127.0.0.1:8080
DirectionLocal only
SourceStreamer.bot
ModeSubscribe → Receive → Execute
AuthNone (localhost boundary)

No internet exposure. No remote access. No funny business.

📡 Subscription Model

Stream Connector subscribes to Streamer.bot’s Custom Events channel. 

{
  "request": "Subscribe",
  "id": "streamconnector-sub",
  "events": {
    "General": ["Custom"]
  }
}

Only events explicitly sent by Streamer.bot are processed.

🧱 Event Schema

All events are normalized before execution:

{
  "provider": "string",
  "commandId": "string",
  "context": { }
}

No eval. No dynamic execution. No surprises.
