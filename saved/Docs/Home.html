The Ultimate Toolkit for VRChat, TikTok, and Real-Time Interactivity

Stream Connector is a powerful interactivity engine designed for creators, VTubers, and streamers who want full control over live reactions, device integrations, and real-time logic.

It connects:

  • TikTok Live
  • Streamer.bot
  • VRChat OSC
  • PiShock devices
  • Intiface / Bluetooth toys
  • OwO haptics
  • External APIs
  • Custom logic chains

All in one unified system.

What Stream Connector Is

Stream Connector is not a plugin or a toy tool.

It is a modular, event-driven engine that lets you:

  • React to live events instantly
  • Control hardware and avatars
  • Drive your devices from live VRChat touch (SPS / OGB)
  • Build logic flows
  • Create chaotic or controlled interactions
  • Run everything locally and securely

Editions

Stream Connector comes in two editions, built from the same engine:

  • Stream Connector - the full toolkit: TikTok Live, Streamer.bot, VRChat OSC, PiShock, Intiface / Bluetooth, OwO, the external API, and the chain system.
  • VixForge Haptics Nexus - a haptics-only edition for VRChat. It keeps VRChat OSC + OSCQuery, SPS / OGB live-touch, Intiface / Buttplug, PiShock, OwO, and the full chain system, but leaves out the livestreaming sources (no TikTok, TikFinity, or Streamer.bot). For people who only want VRChat touch driving real devices - a lighter, focused interface with its own dark-alloy theme.

Both editions share the same chains, device support, and SPS engine; the Haptics Nexus simply hides the streaming layer.

Core Philosophy

  • No locked ecosystems

Built for creators who want real control, not presets.
